HFDS Prototype

Hands-Free Driving System (HFDS)

This project is an interactive prototype of a Level 2 "hands-off" driving feature. The HFDS system allows a vehicle to automatically steer, accelerate, and brake in certain highway conditions. While active, the driver is continuously monitored to ensure they are engaged and ready to resume control.

Core Features Demonstrated:

  • System Activation: Validating preconditions like speed, maps, and lane markings.
  • Driver Monitoring: Escalating warnings for driver inattention (visual, audio, haptic).
  • Fault Tolerance: Safely handing control back to the driver if a system fault occurs.
  • Manual Override: Allowing the driver to seamlessly take control at any time.
